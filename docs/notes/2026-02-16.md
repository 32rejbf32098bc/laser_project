# Laser arrival and initial system setup

## Objective

- Begin assembly of laser triangulation scanning system
- Verify laser operation
- Attempt initial laser plane calibration

---

## Hardware received

- 650nm red line laser module
- Adjustable mount components
- Bench power supply

---

## Initial hardware setup

- Mounted Raspberry Pi Camera Module 3 onto test rig
- Mounted laser module using temporary bracket
- Powered laser using bench PSU
- Verified stable laser operation

---

## Laser observations

- Laser line appeared as closely spaced dots rather than perfectly continuous line
- Line still visually continuous at working distance
- Laser clearly visible in camera images

---

## First laser plane calibration attempt

### Method

- Captured checkerboard calibration images with laser projected onto checkerboard
- Ran laser plane calibration pipeline

---

### Result

- Calibration failed
- Laser plane fitting error extremely high
- Plane orientation visibly incorrect
- Reconstruction severely distorted

---

### Cause identified

Laser extraction was incorrectly detecting:

- Checkerboard edges
- Reflections
- Background features

Instead of isolating only laser pixels

---

## Conclusion

- Initial laser plane calibration unsuccessful
- Laser filtering required before calibration

---

## Next actions

- Implement laser colour filtering
- Isolate laser pixels prior to calibration
- Re-run laser plane calibration
