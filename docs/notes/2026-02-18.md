# Full scan pipeline and reconstruction

## Objective

- Run full scanning pipeline
- Generate first complete 3D point cloud

---

## Scan parameters

- Scan length: 80 mm
- Step size: 0.05 mm
- Camera resolution: 4608 × 2592

---

## Pipeline stages completed

- Image capture
- Laser ridge detection using Steger method
- Triangulation using calibrated laser plane
- Multi-frame point cloud reconstruction

---

## Result

- First full 3D reconstruction successfully generated
- Point cloud clearly showed:

  - Surface geometry
  - Pits
  - Crack features

---

## Major discovery

Steger ridge detector already provides:

- Accurate subpixel laser center points

Centerline post-processing was unnecessary and caused errors

---

## Pipeline improvement

- Removed centerline fitting stage
- Used Steger ridge points directly for triangulation

---

## ROI debugging

Implemented ROI overlay visualisation

Discovered ROI configuration error:

- ROI was excluding sample ridge

Corrected ROI configuration

Result:

- Correct laser detection
- Improved reconstruction quality

---

## Noise investigation

Observed noise caused by:

- Surface reflections
- Background features
- Weak ridge detections

Plan to apply ridge strength filtering

---

## Conclusion

First fully working laser triangulation reconstruction pipeline achieved

System now capable of producing accurate 3D surface measurements

---

## Next actions

- Apply ridge strength filtering
- Reduce noise
- Optimise pipeline performance
- Integrate system onto robot platform
